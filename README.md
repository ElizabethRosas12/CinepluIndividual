CinePlus 

Se describe cómo realizar la ejecución del proyecto de cine, así como también se muestran las pantallas

Inicio Sesion:
se muestra en pantalla al usuario para que se pueda registrar y pueda realizar una compra mas rapida 

Taquilla:
esta pantalla muestra la pelicula a seleccionar, su sala y horarios para que el usuario pueda escoger

Dulceria:
muestra los productos que ofrece el cine para que el usuario pueda escoger con facilidad que productos quiere 

Ticket:
muestra el precio y el total de cada pantalla, asi como todos los productos escogidos para si quiere cancelar o modificar, lo haga

Para poder ejecutar nuestro proyecto “CinePlus” abriremos nuestra carpeta creada con todos los documentos y archivos que hayamos agregado a dicho proyecto, esta carpeta la abriremos en visual studio code, el cual ya tendrá descargado wxglade. En la parte izquierda nos deberán aparecer todos nuestros documentos. Por medio del código ejecutaremos nuestro código realizado y editado para que se muestre todo lo que el usuario podrá hacer en dicho proyecto, como iniciar sesion, así también como seleccionar sus boletos, asientos, horario y cantidad de personas que disfrutarán de nuestras películas; también podrán escoger los combos en nuestra ventana de dulcería creada hacia el usuario, donde mostraremos distintos combos a escoger, productos individuales; así también mostrará una pantalla donde estarán todos los productos que selecciono, en una ventana que se mostrará como ticket, ahí podrá cancelar o aceptar su pedido, este proyecto se mostrará ejecutado gracias a la modificación que se hizo por medio de visual studio code, Python y wxglade.

lenguajes a utilzar

Python, wxglade, Visual Studio Code
